# صانع الإنفوجرافيك الصحفي الذكي (Smart Journalism Infographic Maker)

منصة متقدمة تعتمد على الذكاء الاصطناعي لتحويل المحتوى الصحفي والبيانات الخام إلى إنفوجرافيك احترافي جاهز للنشر. تم تصميم هذه المنصة خصيصاً لتلبية احتياجات غرف الأخبار الحديثة وصناع المحتوى الرقمي.

## 🚀 نظرة عامة
تستخدم المنصة قوة نماذج **Google Gemini 2.5** للقيام بثلاث مهام رئيسية:
1. **التحليل الذكي:** استخراج البيانات والعناوين من المقالات الصحفية الطويلة.
2. **صياغة البرومبت:** تحويل البيانات إلى وصف فني (Prompt) دقيق ومحسّن.
3. **التوليد البصري:** إنتاج صورة إنفوجرافيك نهائية بجودة عالية باستخدام نموذج "Nano Banana".

## ✨ المميزات الرئيسية

### 1. استخراج البيانات من المقالات (Article-to-Data)
بفضل نموذج `gemini-2.5-flash` المستند إلى معالجة اللغات الطبيعية، يمكن للمستخدم لصق مقال كامل، ليقوم النظام تلقائياً بـ:
- اختيار نوع الإنفوجرافيك الأنسب.
- صياغة عنوان جذاب.
- استخلاص الأرقام والبيانات في جدول منظم.

### 2. أنواع إنفوجرافيك متعددة (9 أنواع)
- **إحصائي (Statistical):** للتركيز على الأرقام والنسب.
- **زمني (Timeline):** لسرد الأحداث التاريخية.
- **جغرافي (Geographical):** للخرائط والتوزيع المكاني.
- **عمليات (Process):** لشرح خطوات العمل.
- **تشريحي (Anatomy):** لتفكيك أجزاء النظم أو الكائنات.
- **مقارنة (Comparison):** للمقارنة بين كيانين.
- **قوائم (Listicle):** لتبسيط المعلومات في نقاط.
- **سيرة ذاتية (Bio Profile):** للتعريف بالشخصيات.
- **لوحة بيانات (Dashboard):** لعرض شامل ومعقد للبيانات.

### 3. أنماط بصرية احترافية (Themes)
- **Dark Modern:** واجهات داكنة عصرية.
- **Clean White:** نمط المطبوعات النظيف.
- **Tech Futuristic:** لمسة تقنية ومستقبلية.
- **Retro Vintage:** طابع كلاسيكي وثائقي.
- **Minimalist:** بساطة تامة وتركيز عالي.
- **Isometric 3D:** عمق بصري ثلاثي الأبعاد.

### 4. التحكم في الأبعاد (Aspect Ratios)
- **1:1 (Square):** مثالي لإنستغرام وفيسبوك.
- **16:9 (Landscape):** مثالي للمواقع الإلكترونية وتويتر.
- **9:16 (Portrait):** مثالي للقصص (Stories) وتيك توك.

## 🛠 التقنيات المستخدمة
- **Frontend:** React 19, TypeScript.
- **Styling:** Tailwind CSS.
- **Icons:** Lucide-React.
- **AI Engine:** `@google/genai` (Google Gemini API).
  - `gemini-2.5-flash`: للتحليل واستخراج الـ JSON.
  - `gemini-2.5-pro`: لصياغة البرومبتات الفنية.
  - `gemini-2.5-flash-image`: لتوليد الصور النهائية.

## 📖 كيفية الاستخدام
1. **التحليل:** اضغط على "استخراج البيانات من مقال" والصق نص خبرك.
2. **المراجعة:** راجع البيانات المستخرجة في الجدول وقم بتعديلها إذا لزم الأمر.
3. **التخصيص:** اختر النمط البصري (Theme) والأبعاد المناسبة لمنصتك.
4. **التوليد:** اضغط على "توليد البرومبت" ثم "معاينة الإنفوجرافيك".
5. **التحميل:** بعد ظهور الصورة، يمكنك تحميلها مباشرة عبر زر التحميل.
